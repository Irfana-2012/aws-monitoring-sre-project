# Enterprise Monitoring & Auto Scaling AWS Project

## Technologies Used
- AWS EC2
- Docker
- Prometheus
- Grafana
- CloudWatch
- Load Balancer
- Auto Scaling
- Linux
- Flask

- Dockerized Flask application

- Auto Scaling Group

- Production-style deployment
